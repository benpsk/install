# PHP Stack Server Deployment Script

### Available Services

- PHP 
  - v8.2
  - v8.1
  - v8.0
  - v7.4
  - v7.2
- Composer
- Database 
  - MySQL 
  - MariaDB
- Web Server 
  - Nginx 
  - Apache2

### All you need to do is - 

1. Download the ipWizard.sh.
```sh
wget https://raw.githubusercontent.com/benpsk/install-php/main/ipWizard.sh
```

2. Run the script.
```sh
sudo bash ipWizard.sh
```
